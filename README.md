This project demonstrates a complete integration between a Node.js/Express backend and TallyPrime (India's leading accounting software) using TallyPrime's native HTTP XML interface on port 9000. TallyPrime runs on an AWS EC2 Windows Server instance, accessed remotely via RDP, ensuring 24/7 uptime and a stable public IP. The application can fetch company data, retrieve ledgers, and push sales invoices directly into TallyPrime — with every transaction persisted to a PostgreSQL database for audit and tracking. No third-party middleware or additional Tally licensing is required.
